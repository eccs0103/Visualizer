Settings

Loop audio

Enables audio repeat.

Autoplay

Autoplay on audio uploading.

Information

Duration of information display. 0 means hidden.
If the name of the uploaded file is in the form "author - title", then the program will display that information. Value is in seconds.

Visualization

The visualization. Appearance, animations, rendering when playing music.
This is more suitable for calm songs where the beat is not prominent and/or the music is primarily instrumental. This is more suitable for active songs where the beat is distinct and/or loud, separated music is playing.

Quality

Audio processing quality. Recommended level 10. Levels before 8 and after 12 are not recommended.

Smoothing

A smoother set of visualizations changes over time. In the range from 0 to 1. 0 changes instantly, 1 changes with great smoothness. Values before 0.1 and after 0.9 are not recommended.

Limiters

Sound range limiters. Values are in decibels.

Anchor

Anchor between visualization and reflection in relation to page height.
